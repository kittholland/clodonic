# Rails Development Rules ## Testing Requirements - Minimum 90% test coverage - All PRs must include tests - Use RSpec for all tests - FactoryBot for test data ## Code Style - Follow Ruby Style Guide - Use Rubocop with our config - Max line length: 100 chars ## Database - All migrations reversible - Add indexes for foreign keys - Use strong_migrations gem ## Performance - N+1 queries forbidden - Use includes() for associations - Cache expensive operations
